Gibberish + Interface = Gibberface

To experiment with Gibberface try the Gibberface playground. Alternatively, below are a bunch of examples of Gibberface in action. Gibberface is made using Gibberish.js and Interface.js. You can grab the Gibberface repo.

Simple

  • Sliders + Sine - Use sliders to control the pitch and amplitude of a sine wave.
  • Gyro + Sine - (Safari Mobile or Chrome only) Use the Gyro sensor to control the pitch of a sine wave.
  • Buttons + Synth - Use Buttons to play notes on a simple synth with an envelope.
  • Buttons + Sliders + Synth - Use Buttons to play notes on a simple synth. Use sliders to change the synths attack, decay and amplitude.
  • Buttons + Knobs + Synth2 - Use Buttons to play notes on a simple synth. Use sliders to control filter cutoff, filter resonance and amp.

Trickier

  • Gyro + Button + Sampler + Flanger - (Safari Mobile or Chrome only) Use the Gyro sensor to control the pitch of sample playback and the feedback amount for a flanger. Use a slider to control the modulation speed of the flanger.
  • RangeSlider + Sliders + Sampler - Use a Range slider to select the portion of an audio file to play, use sliders to control delay and filter characteristics.
  • XY + Sliders + PWM + Filter - Use XY controller to contol pitch and pulsewidth of oscillators. Use sliders to control filter cutoff, amplitude, and friction of XY widget.

Kind of fancy

  • Drum Sequencer - MultiButton + Samplers + FX - A simple four-part drum sequencer with some fx to play with.
